# Seite „Dateien auswählen“


Auf der Seite **Dateien auswählen** können Sie die Dateien überprüfen, die mit dem Paket Beschleuniger gespeichert werden. Einige dieser Dateien werden angezeigt, weil Sie sich nicht auf dem angegebenen Medium oder auf dem lokalen Computer befanden. In einigen Szenarien sollten die angezeigten Dateien mit dem Paket Beschleuniger gespeichert werden, damit das virtuelle Anwendungspaket beim Anwenden des Paket Beschleunigers vollständig neu erstellt werden kann.

Diese Seite enthält die folgenden Elemente:

<a href="" id="select-files-pane"></a>**Bereich ' Dateien auswählen '**  
Zeigt eine Liste der Dateien an, die im Paket Beschleuniger gefunden wurden. Überprüfen Sie die Dateien sorgfältig, und löschen Sie eine Datei, die aus dem Paket Beschleuniger entfernt werden soll. Wählen Sie nur Dateien aus, die für die erfolgreiche Ausführung der Anwendung erforderlich sind.
