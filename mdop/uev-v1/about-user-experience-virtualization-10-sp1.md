## Office 2007-Einstellungen, Speicherort Vorlagen


Die Installationssoftware des UE-V-Agents installiert den Agenten und registriert eine Standardgruppe von Einstellungen-Standort Vorlagen für gängige Microsoft-Anwendungen. Microsoft Office 2007 ist jetzt Teil dieser Anwendungen. Es gibt zwei Office 2007-Vorlagen: MicrosoftOffice2007.xml und MicrosoftCommunicator2007.xml. Mit den Einstellungen für Standort Vorlagen werden die Einstellungen in Microsoft Office 2007 für die folgenden Anwendungen erfasst:

-   Microsoft Access 2007

-   Microsoft Communicator 2007

-   Microsoft Excel 2007

-   Microsoft InfoPath 2007

-   Microsoft OneNote 2007

-   Microsoft Outlook 2007

-   Microsoft PowerPoint 2007

-   Microsoft Project 2007

-   Microsoft Publisher 2007

-   Microsoft SharePoint Designer 2007

-   Microsoft Visio 2007

-   Microsoft Word 2007

### Updates für Office 2010-Einstellungen für Standort Vorlagen

Außerdem wurde eine Aktualisierung der Einstellungen-Standort Vorlagen durchgeführt. Zu diesen Änderungen gehören:

-   Unterstützung für Microsoft SharePoint Designer 2010 durch Hinzufügen einer neuen Vorlage zu den Office 2010-Vorlagen (MicrosoftOffice2010Win32.xml und MicrosoftOffice2010Win64.xml)

-   Kleinere Fehlerbehebungen, einschließlich Anpassen der Status Leiste – Word, Excel und PowerPoint

## Geplanter Vorgang für Katalog Updates ist jetzt randomisiert


Mit dem automatischen Aktualisierungsvorgang der Vorlage wird der Einstellungs Vorlagenkatalog auf neue, aktualisierte oder entfernte Vorlagen überprüft. Diese Aufgabe wird nur ausgeführt, wenn die SettingsTemplateCatalog konfiguriert ist. Der Task "automatische Aktualisierung der Vorlage" führt die ApplySettingsCatalog.exe-Datei aus, die sich im Installationsverzeichnis des UE-v-Agents befindet und mit UE-v SP1 geändert wurde, um das Update über einen Zeitraum von einer Stunde zu randomizen.

## Unterstützung für Citrix EdgeSight


Es wurde ein Konflikt festgestellt, bei dem UE-V auf einem Server mit Citrix EdgeSight ausgeführt wurde. UE-V 1,0 SP1 behebt dieses Problem.

## Indizierung von Internet Explorer-Favoriten


Wenn UE-V Internet Explorer-Favoriten von einem Computer auf einen anderen Computer durchstreift, wird die Indizierung der bevorzugten Adressen in der Adressleiste auf dem synchronisierten Computer nun aktualisiert. Wenn ein Benutzer die Adressleiste eingibt, werden die Roaming-Favoriten nun auf synchronisierten Computern als verfügbares Suchergebnis angezeigt.

## Neue setup.exe Befehlszeilenparameter für UE-v-Agent und UE-v-Generator


Mit der Veröffentlichung von UE-v 1,0 SP1 wurden die setup.exe für den UE-v-Agent und den UE-v-Generator aktualisiert, um die folgenden zusätzlichen Befehlszeilenparameter zu ermöglichen:

1.  `CEIPENABLED` – Ermöglicht es Setup, die Option zu akzeptieren, die in das Programm zur Verbesserung der Benutzerfreundlichkeit von Microsoft aufgenommen werden soll.

2.  `INSTALLFOLDER` – Ermöglicht das Einrichten eines anderen Installationsordners für den Agenten oder Generator.

3.  `MUENABLED` – Ermöglicht es Setup, die Option zu akzeptieren, die im Microsoft Update-Programm enthalten sein soll.
