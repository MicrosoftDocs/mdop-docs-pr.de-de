# Seite „Task auswählen“ (Weitere Informationen)


Verwenden Sie die Seite **Aufgabe auswählen** , um ein vorhandenes virtuelles Anwendungspaket zu ändern. Weitere Informationen zum Ändern eines vorhandenen virtuellen Anwendungspakets finden Sie unter [so wird es gemacht: Ändern eines vorhandenen virtuellen Anwendungspakets (App-V 4,6 SP1)](how-to-modify-an-existing-virtual-application-package--app-v-46-sp1-.md).

Diese Seite enthält die folgenden Elemente:

<a href="" id="update-application-in-existing-package"></a>**Anwendung in bestehendem Paket aktualisieren**  
Wählen Sie diese Option aus, um ein Update auf eine Anwendung oder ein Programm anzuwenden, das Teil eines vorhandenen virtuellen Anwendungspakets ist. Wenn Sie diese Option auswählen, sollten die zugehörigen Installationsdateien lokal auf dem Computer gespeichert sein, auf dem der App-V-Sequencer ausgeführt wird. Sie müssen auch auf den Speicherort zugreifen können, an dem das Paket, das die Anwendung oder das Programm enthält, die Sie ändern möchten, gespeichert ist.

<a href="" id="edit-package"></a>**Paket bearbeiten**  
Wählen Sie diese Option aus, um die Eigenschaften zu ändern, die einem vorhandenen virtuellen Anwendungspaket zugeordnet sind. Zum Bearbeiten eines Pakets müssen Sie auf den Speicherort zugreifen können, an dem das virtuelle Anwendungspaket gespeichert ist. In der folgenden Liste werden die Paketeigenschaften angezeigt, die aktualisiert werden können, wenn Sie **Paket bearbeiten**auswählen:

-   Anzeigen von Paketeigenschaften.

-   Anzeigen des Paket Änderungsverlaufs

-   Anzeigen zugehöriger Paketdateien

-   Bearbeiten Sie die Registrierungseinstellungen.

-   Überprüfen Sie die zusätzlichen Paketeinstellungen (mit Ausnahme der Dateieigenschaften des Betriebssystems).

-   Erstellen eines zugehörigen Windows Installers

-   OSD-Datei ändern.

-   Komprimieren und Dekomprimieren des Pakets.

-   Dateitypzuordnungen hinzufügen.

-   Umbenennen von Tastenkombinationen

-   Legen Sie den virtualisierten Registrierungsschlüssel Status fest (override/Merge).

-   Legen Sie den Status eines virtualisierten Ordners fest.

-   Bearbeiten von virtuellen Dateisystem Zuordnungen

<a href="" id="add-new-application"></a>**Neue Anwendung hinzufügen**  
Wählen Sie diese Option aus, um einem vorhandenen virtuellen Anwendungspaket eine neue Anwendung oder ein neues Programm hinzuzufügen. So können Sie beispielsweise Microsoft Excel zu einem vorhandenen virtuellen Microsoft Office-Anwendungspaket hinzufügen. Wenn Sie eine neue Anwendung hinzufügen möchten, müssen Sie auf den Speicherort zugreifen können, an dem das virtuelle Anwendungspaket gespeichert ist. Sie können auch Dateitypzuordnungen bearbeiten, die mit einem vorhandenen Paket verknüpft sind.
