# So entfernen Sie eine Anwendung aus einer Lizenzgruppe


Sie können das folgende Verfahren verwenden, um eine Anwendung aus der zugewiesenen Lizenzgruppe in der Application Virtualization Server-Verwaltungskonsole zu entfernen. Anschließend können Sie keine Lizenzsteuerelemente anwenden oder Berichte verwenden, um die Lizenzverwendung für diese Anwendung zu überwachen.

**So entfernen Sie eine Anwendung aus einer Lizenzgruppe**

1.  Klicken Sie im linken Bereich der Verwaltungskonsole auf den Knoten **Anwendungen** , und erweitern Sie ihn.

2.  Klicken Sie mit der rechten Maustaste auf die gewünschte Anwendung, und wählen Sie **Eigenschaften**aus.

3.  Verwenden Sie auf der Registerkarte **Allgemein** das Pulldown-Menü **Anwendungslizenzgruppe** , um ** &lt; keine &gt; **auszuwählen.

4.  Klicken Sie auf **OK**.

    **Hinweis**  Sie können die Registerkarte **Eigenschaften** einer Anwendung gleichzeitig ändern.
