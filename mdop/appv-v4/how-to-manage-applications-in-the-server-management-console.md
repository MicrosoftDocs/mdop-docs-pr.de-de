# So verwalten Sie Anwendungen in der Server Management Console


In der Application Virtualization Server-Verwaltungskonsole können Sie alle Anwendungen für einen oder mehrere Application Virtualization-Server hinzufügen, entfernen, konfigurieren und steuern.

## Inhalt dieses Abschnitts


<a href="" id="how-to-import-an-application"></a>[So importieren Sie eine Anwendung](how-to-import-an-applicationserver.md)  
Enthält Schritt-für-Schritt-Anweisungen zum Importieren einer Anwendung durch Importieren der Datei "Open Software Descriptor (OSD)" oder "Sequencer Project (SPRJ)".

<a href="" id="how-to-rename-an-application"></a>[So benennen Sie eine Anwendung um](how-to-rename-an-application.md)  
Enthält Schritt-für-Schritt-Anweisungen zum Umbenennen einer vorhandenen Anwendung.

<a href="" id="how-to-delete-an-application"></a>[So löschen Sie eine Anwendung](how-to-delete-an-application-server.md)  
Enthält Schritt-für-Schritt-Anweisungen zum Entfernen oder Löschen einer Anwendung.

<a href="" id="how-to-manually-add-an-application"></a>[So fügen Sie eine Anwendung manuell hinzu](how-to-manually-add-an-application.md)  
Enthält Schritt-für-Schritt-Anweisungen zum manuellen Hinzufügen einer Anwendung zu den Servern.

<a href="" id="how-to-move-an-application"></a>[So verschieben Sie eine Anwendung](how-to-move-an-application.md)  
Enthält Schritt-für-Schritt-Anleitungen zum Verschieben einer Anwendung aus einer Anwendungsgruppe in eine andere Anwendungsgruppe.

<a href="" id="how-to-grant-access-to-an-application"></a>[So gewähren Sie Zugriff auf eine Anwendung](how-to-grant-access-to-an-application.md)  
Bietet Schritt-für-Schritt-Anleitungen, wie Sie Benutzern Zugriff auf Anwendungen erteilen können.

<a href="" id="how-to-deny-access-to-an-application"></a>[So verweigern Sie den Zugriff auf eine Anwendung](how-to-deny-access-to-an-application.md)  
Bietet Schritt-für-Schritt-Anleitungen zum Verweigern der Zugriffsberechtigung für Benutzer auf Anwendungen.

<a href="" id="how-to-change-an-application-icon"></a>[So ändern Sie das Anwendungssymbol](how-to-change-an-application-iconserver.md)  
Enthält Schritt-für-Schritt-Anweisungen zum Ändern des Anwendungssymbols.
